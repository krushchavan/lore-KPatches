This patch allows the firmware_class.path to accept multiple paths separated by a colon (:). It introduces a new helper function fw_try_firmware_path() that tries each path in order before falling back to default search paths. The patch also adds support for escaping literal colons and backslashes using a backslash ("). This change enables users to specify custom firmware directories via the kernel command line, improving flexibility and usability.

Neutral llm-chunked-no-replies Awaiting Review

This patch adds a new netlink command, NFSD_CMD_CACHE_FLUSH, which allows userspace to flush the nfsd export caches without writing to /proc/net/rpc/*/flush. The command takes an optional attribute, NFSD_A_CACHE_FLUSH_MASK, that selects which caches to flush. If omitted, all nfsd caches are flushed. This is used by exportfs to replace its /proc-based cache_flush() with a netlink equivalent, with /proc fallback for older kernels.

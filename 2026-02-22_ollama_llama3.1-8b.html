Ongoing Submitted 2026-02-20 [PATCH 0/3] sunrpc: cache infrastructure scalability improvements 3 patches Needs Work LLM (per-reviewer) Accepted

This patch series improves the scalability of the sunrpc cache infrastructure by converting a global spinlock and waitqueue to per-cache-detail locks and waitqueues, allowing for more efficient concurrent access. The patches also split the cache detail queue into separate lists for readers and requests, simplifying the code and enabling future netlink upcall implementation. This change is intended for Linux kernel version 7.1.

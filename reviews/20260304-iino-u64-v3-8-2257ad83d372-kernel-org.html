[PATCH v3 08/12] zonefs: widen trace event i_ino fields to u64

View on lore.kernel.org →
Active on: 2026-03-04
Patch summary
This patch updates the zonefs trace event definitions to use u64 instead of ino_t/unsigned long for inode number fields, allowing for wider range and more precise tracking of inode numbers.
Jeff Linux-f2fs-devel 2026-03-04 Reviewed-by Neutral LLM

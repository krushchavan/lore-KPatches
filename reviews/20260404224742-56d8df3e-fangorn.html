Patch summary
This patch fixes a use-after-free bug in the CPU accounting subsystem of the Linux kernel. The bug occurs when a timer interrupt on one CPU accesses data from another CPU's cpuacct structure, which has already been freed by the free_percpu() function. To fix this, the patch introduces an RCU callback to defer the actual freeing of percpu data and the cpuacct struct until all concurrent readers have completed their operations.

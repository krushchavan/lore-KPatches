Patch summary
This patch adds a new function `sdei_is_present()` to the Linux kernel, which checks whether SDEI firmware is present and usable. The function returns a boolean value indicating whether SDEI firmware has been probed and is available for use.
Kiryl Shutsemau 2026-06-30 Neutral LLM
Agreed to move the function to patch 3 if the maintainer prefers it, but expressed no strong opinion on the matter. Also mentioned that checkpatch's preference for `!= NULL` is a minor issue.

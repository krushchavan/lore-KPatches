Patch summary
This patch adds support for zswap and zeroswap as virtual swap backends, allowing pages to be compressed or detected as zero-filled without pre-allocating physical swap slots. It builds on top of the existing swap-table infrastructure and introduces virtual swap entries that decouple PTE swap entries from physical backing. The patch only supports zswap and zero-page backends for now, with physical disk backing fallback and writeback support to be added in subsequent patches.

Patch summary
This patch series adds support for buffer rings and zero-copy to the FUSE (Filesystem in Userspace) kernel module, utilizing io-uring capabilities. Buffer rings reduce memory usage by pooling payload buffers and selecting them on demand, while also laying the groundwork for pinned buffers that can be easily managed. Zero-copy is an opt-in feature for privileged servers that eliminates memory copies between the kernel and userspace for read/write-heavy operations. The patch series has been tested to show throughput improvements of up to 35% in certain scenarios.

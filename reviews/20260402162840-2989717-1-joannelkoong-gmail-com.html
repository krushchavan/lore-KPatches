[PATCH v2 00/14] fuse: add io-uring buffer rings and zero-copy

View on lore.kernel.org →
Submitted: 2026-04-02
Active on: 2026-04-02
Patch summary
This patch series adds buffer ring and zero-copy capabilities to the FUSE file system over io-uring, reducing memory usage and enabling pinned buffers for future optimizations.
Joanne Koong 2026-04-02 Neutral LLM
Moved fuse_uring_abort() out of the inline header definition and into dev_uring.c to make subsequent commits cleaner.
Show original comment 
Simplify the logic for fetching + sending off the next request.

This gets rid of fuse_uring_send_next_to_ring() which contained
duplicated logic from fuse_uring_send(). This decouples request fetching
from the send operation, which makes the control flow clearer and
reduces unnecessary parameter passing.

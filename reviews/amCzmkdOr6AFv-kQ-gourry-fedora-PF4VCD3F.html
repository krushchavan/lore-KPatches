Patch summary
This patch modifies DAMON (Dynamic Access Monitor) to skip memory from private nodes during migration and pageout operations. This change ensures that user-controlled MADV commands for private node folios behave similarly to ZONE_DEVICE folios, which are already handled correctly by DAMON. The patch addresses a potential issue where DAMON might not handle zone device pages properly.

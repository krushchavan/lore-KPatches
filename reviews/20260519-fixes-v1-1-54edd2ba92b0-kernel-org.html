Patch summary
This patch fixes issues in Ansible playbooks caused by changes to boolean handling in ansible-core 2.20 and later. Specifically, it addresses problems where variables were being treated as strings instead of booleans, leading to fatal errors or incorrect behavior. The patch updates various playbook files to use the correct syntax for evaluating booleans and converts string variables to proper booleans using filters such as |bool and |default(true). This ensures that Ansible playbooks continue to function correctly with the updated core version.

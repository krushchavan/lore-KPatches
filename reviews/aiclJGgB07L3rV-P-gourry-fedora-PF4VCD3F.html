Patch summary
This patch is part of a larger series that aims to improve memory management in the Linux kernel. Specifically, it uses the folio_zero_user function for user pages in the post_alloc_hook. The goal is to make the code more efficient and easier to understand by breaking down complex changes into smaller, more manageable pieces. The patch set has become unwieldy due to its complexity and scope, so Gregory suggests splitting it up into smaller pull-ahead sets that can be reviewed separately. This patch is one of the core improvements that will eventually lead to a final feature in multiple kernel versions.

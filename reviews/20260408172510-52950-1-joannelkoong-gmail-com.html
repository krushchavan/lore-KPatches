[PATCH v2] fuse: fix io-uring background queue dispatch on request completion

View on lore.kernel.org →
Submitted: 2026-04-16
Active on: 2026-04-16
Patch summary
This patch fixes an issue in the FUSE (Filesystem in Userspace) implementation where background requests completed via io-uring were not properly dispatched due to incorrect handling of connection-level background counters. The fix involves accounting for these counters before flushing the queue's background queue, ensuring that pending background requests are dispatched correctly. This patch addresses a problem introduced by a previous commit that allowed bg requests to be queued through io-uring, and it has been reviewed and tested to ensure correctness.

David Laight 2026-02-23 Inline Review Reviewed-by Needs Work LLM
Reviewer noted that the O_EMPTY_PATH flag is not properly handled in the build_open_flags function, which does not set LOOKUP_EMPTY when O_EMPTY_PATH is specified. Reviewer David Laight questioned the usefulness of allowing empty path in openat and openat2 syscalls for files, citing security concerns and lack of clear use cases.

[PATCH v2 20/21] fuse: use passthrough getattr in setattr suid/sgid handling

Active on: 2026-05-17 • 2026-05-16
Patch summary
This patch series modifies the FUSE (Filesystem in Userspace) kernel module to handle suid/sgid permissions when setting attributes on files. The goal is to ensure that these permissions are correctly propagated from the server to the client, even if the getattr operation is passed through.
Amir Goldstein 2026-05-17 Neutral LLM
Raised the possibility that for getattr operations related to killpriv, it may make sense to ask the server's mode for calculating the setattr mode. However, did not express a strong opinion or request changes.

Patch summary
This patch modifies the way memory cgroup (memcg) caches object stock, switching from caching by object color group pointer to caching by memcg. This change aims to address a significant performance regression introduced in a previous version of the kernel, which resulted in a 67% slowdown on certain systems. The solution is expected to be fully implemented within one more development cycle.

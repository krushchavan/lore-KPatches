Patch summary
This patch adds an interface for selecting a swap tier, which is a way to control memory allocation between different types of storage. The current implementation makes tiered memcg limits a cgroup mount option, but does not allow individual memcgs to tune their limits or opt-in/out. The author suggests that making these values tunable could be considered in the future, potentially aligning with existing file names and interaction methods for memory allocation.

Patch summary
This patch set addresses issues related to delegated attributes in the NFS file system, specifically fixing problems with utimensat() and removexattr operations. The patches update inode timestamps and ctime after attribute changes, ensuring accurate and consistent behavior. The fixes are based on a previous version of the patch set, which has been updated with Fixes: tags for better tracking.

She also pointed out confusion about the difference between time_access and time_modify values in SETATTR.

[PATCH v2 073/110] befs: replace PRIino with %llu/%llx format strings

View on lore.kernel.org →
Submitted: 2026-03-02
Active on: 2026-03-02
Patch summary
This patch updates the befs file system to use the correct format strings for printing inode numbers, which are now u64 type. The PRIino macro has been removed and replaced with %llu/%llx in all relevant places. This ensures that inode numbers are printed correctly and consistently throughout the code.

Patch summary
This patch adds a new extended entry reply structure called fuse_entry2_out, which carries additional information such as backing_id and statx attributes. It also introduces helpers to process this new structure for passthrough support in lookup, revalidate, and create operations. The existing fuse_statx_to_attr function is moved earlier in the file to avoid forward declarations.

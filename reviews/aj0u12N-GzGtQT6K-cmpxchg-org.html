Patch summary
This patch modifies the Linux kernel's memory cgroup charging mechanism to avoid splitting large folios when there is no available swap space. The current implementation tries to charge against the first ancestor that fails, but this can lead to unnecessary splits. Instead, the patch proposes calculating the actual headroom by walking up the hierarchy and taking the minimum of all ancestors' available swap space. This approach aims to relax the strict limit enforcement in favor of a more flexible margin-based calculation.

Reviewer questioned the necessity of the patch, suggesting that a fuzz factor around the limit is acceptable and noting that the remaining headroom is less than a large folio. Reviewer Barry Song noted that the current implementation may incur significant latency for I/O or compression, and suggested splitting huge folios into 2MB folios instead of base pages when swapping out to allow architectures with large folios to use THP_SWAP. Reviewer pointed out that the patch's approach is similar to what mem_cgroup_get_nr_swap_pages() already does, suggesting that the effort may not be worth it given the limited headroom. Reviewer Barry Song suggested modifying the __mem_cgroup_try_charge_swap() function to return an additional argument for the remaining swap space, which would be used in conjunction with mem_cgroup_margin() math to determine headroom. He proposed adding a fuzz factor around the limit and provided a code concept that includes this modification.

It seems it is exactly what mem_cgroup_get_nr_swap_pages() is doing:

long mem_cgroup_get_nr_swap_pages(struct mem_cgroup *memcg)
{
        long nr_swap_pages = get_nr_swap_pages();

        if (mem_cgroup_disabled() || do_memsw_account())
                return nr_swap_pages;
        for (; !mem_cgroup_is_root(memcg); memcg = parent_mem_cgroup(memcg))
                nr_swap_pages = min_t(long, nr_swap_pages,
                                      READ_ONCE(memcg->swap.max) -
                                      page_counter_read(&memcg->swap));
        return nr_swap_pages;
}

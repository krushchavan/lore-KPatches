[PATCH v2 108/110] net: replace PRIino with %llu/%llx format strings

View on lore.kernel.org →
Active on: 2026-03-02
Patch summary
This patch replaces the PRIino format macro with concrete format strings %llu and %llx in several network-related files, including net/netrom/af_netrom.c, net/rose/af_rose.c, and net/socket.c. The change is necessary because i_ino is now a u64 type, and PRIino has been removed. This patch ensures that the correct format strings are used to print inode numbers in these files.
Jeff Layton (author) 2026-03-02 Neutral LLM
The author is addressing a concern about the PRIino format macro being removed and replaced with concrete format strings, specifically that the %llu/%llx format strings should be used instead of %llu for u64 types.
Show original comment 
Now that i_ino is u64 and the PRIino format macro has been removed,
replace all uses in net with the concrete format strings.

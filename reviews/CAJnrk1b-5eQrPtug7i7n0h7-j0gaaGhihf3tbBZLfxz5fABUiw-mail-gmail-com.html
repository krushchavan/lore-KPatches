Patch summary
This patch series introduces MGLRU-FG and refault distance support to the Linux kernel's memory management system, improving performance by optimizing page replacement decisions. The changes include a new flag format rework, a generic LRU-refs API, and modifications to working set and referenced flags. The patches are currently being reviewed and discussed, with some suggestions for splitting the series into independent chunks to facilitate review and testing.

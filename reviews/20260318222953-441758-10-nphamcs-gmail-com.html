Patch summary
This patch introduces a new virtual swap slot design, where each swapped out page is dynamically allocated a virtual swap slot and associated metadata structure. The virtual swap slot replaces the physical swap slot in page table entries and is used to index into swap data structures such as the swap cache and zswap tree. The patch also repurposes the swap table infrastructure as a reverse map to look up the virtual swap slot from its associated physical swap slot on the swapfile. This change improves the efficiency of swap operations, including cluster readahead and swap file reclamation.

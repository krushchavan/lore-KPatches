Patch summary
This patch series is a preparatory step for introducing PMD level swap entries in the Linux kernel. It cleans up existing code by adding helper functions, refactoring open-coded logic, and defensive checks to preserve current behavior. The patches do not introduce new functionality but rather lay the groundwork for future changes. They are safe to merge independently and will be followed by a series that introduces PMD swap entries. The approach taken is to simplify and modularize existing code to make it easier to review and maintain.

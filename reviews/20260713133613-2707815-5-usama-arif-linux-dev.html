[PATCH v4 04/11] mm: zswap: add range lookup for large-folio swapin

View on lore.kernel.org →
Submitted: 2026-07-13
Active on: 2026-07-13
Patch summary
This patch adds range lookup functionality to the zswap module, allowing it to handle large folios by scanning the covered range and checking if any slots are still in zswap. If a slot is found, the read operation fails so that the caller can fall back to per-page swapin. The patch also introduces a new function `zswap_range_has_entry` for PMD swap-entry consumers to make the same range decision before attempting PMD-order swapin.

Patch summary
This patch series adds support for netlink upcalls to the NFS daemon (nfsd) and sunrpc cache mechanism, replacing antiquated interfaces in /proc. The new netlink-based interfaces allow for easier extension of export options and better container compatibility. A companion userland patch series is also included to implement these changes in mountd/exportd and exportfs.

+		err = -ENOMEM;

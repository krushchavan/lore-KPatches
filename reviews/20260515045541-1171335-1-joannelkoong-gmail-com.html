Patch summary
This patch series fixes several issues in the FUSE (Filesystem in Userspace) implementation for io-uring, a Linux I/O interface. The patches address races between fuse ring creation and abort, registration and abort, and moving cancelled entries to an in-userspace list. These issues can cause indefinite hangs during unmount and resource leaks. The approach taken involves modifying the FUSE code to ensure that request flushing always happens when cancelling operations.

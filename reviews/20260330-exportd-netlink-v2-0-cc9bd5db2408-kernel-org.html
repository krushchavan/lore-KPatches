Patch summary
This patch series adds support for netlink-based upcalls and downcalls in mountd, exportd, and exportfs, allowing them to use the new netlink interface instead of relying on /proc. This change is part of a larger effort to modernize these components and improve their reliability. The patches also include bugfixes, cleanups, and documentation updates for existing code. Netlink support is made optional with the addition of a --no-netlink option, allowing users to disable it if needed.

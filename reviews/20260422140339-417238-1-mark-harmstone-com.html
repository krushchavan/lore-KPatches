Patch summary
This patch addresses a performance issue in the Btrfs file system where DIO (Direct I/O) writes are being serialized, leading to a significant drop-off in write performance. The problem was observed on a VM with 8 cores and 8GB of RAM, where XFS and ext4 showed much better performance (~3GB/s). To fix this issue, the patch aims to prevent DIO writes from being unnecessarily serialized, allowing for improved write throughput.

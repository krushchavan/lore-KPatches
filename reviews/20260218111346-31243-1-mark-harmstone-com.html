[PATCH] btrfs: fix error messages in btrfs_check_features()

View on lore.kernel.org →
Active on: 2026-02-23 • 2026-02-19 • 2026-02-18
Patch summary
This patch fixes a regression introduced by commit d7f67ac9 in btrfs_check_features() by reverting to the previous behavior of only printing unrecognized flags.
Qu Wenruo 2026-02-18 Reviewed-by Positive LLM
Reviewed the patch and found it to be correct. The error handling in btrfs_check_features() now behaves as expected, only printing unrecognized flags.

Patch summary
This patch proposes a new way to implement ghost swapfiles, allowing for backend switching via Redirect entries. The approach aims to minimize changes to the existing data structures and operations, keeping them as close to the current implementation as possible. This is in contrast to an alternative proposal that would require significant modifications to the swap cluster and table data structures.

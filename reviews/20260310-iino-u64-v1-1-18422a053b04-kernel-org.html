[PATCH 1/2] nilfs2: fix 64-bit division operations in nilfs_bmap_find_target_in_group()

View on lore.kernel.org →
Submitted: 2026-03-10
Active on: 2026-03-10
Patch summary
This patch series fixes a build failure on 32-bit ARM caused by the introduction of u64 inode->i_ino, which requires 64-bit division operations. The fix involves using the 64-bit division interfaces in nilfs_bmap_find_target_in_group().
Ryusuke Konishi 2026-03-10 Acked-by Neutral LLM
Acknowledged the conversion to 64-bit division operations as reasonable.

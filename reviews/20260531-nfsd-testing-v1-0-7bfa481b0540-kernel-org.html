Patch summary
This patch series addresses medium-severity bugs in the NFS server code, including memory leaks and refcounting issues that can be triggered remotely. The patches fix problems such as file handle verification, path reference management, and layout fence worker synchronization. The changes aim to prevent potential buffer overruns and improve overall stability of the NFS server.

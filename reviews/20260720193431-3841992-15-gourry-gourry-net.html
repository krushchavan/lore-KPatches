Patch summary
This patch modifies the DAMON (Data Access Monitor) kernel module to skip private node memory during migration and pageout operations. Private node memory is not reclaimable by default, so this change ensures that DAMON does not attempt to migrate or reclaim such memory. The modification involves adding checks for private node memory in both the migration and pageout paths of the DAMON code.

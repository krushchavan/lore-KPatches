Patch summary
This patch adds a benchmark suite to test the performance of NFSD I/O modes using fio and the libnfs backend against an NFS server on localhost. The suite includes tests for buffered, direct, and cached I/O modes via NFSD debugfs controls. It also includes scripts to orchestrate the test matrix, parse results from fio JSON output, and configure the NFS export for testing. This patch aims to provide a comprehensive benchmarking tool for evaluating NFSD performance in various scenarios.

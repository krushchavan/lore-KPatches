Patch summary
This patch adds a new test to the server41tests suite, specifically designed to verify that a directory delegation recall is triggered when renaming a file from another client. The test involves creating a file, obtaining a dir delegation, and then renaming the file from a second client, which should trigger the recall.
Scott Mayhew 2026-04-16 Neutral LLM
Recommended closing the file at the end of the test to avoid leaving state behind on the server.

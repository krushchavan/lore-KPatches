Ongoing Submitted 2026-01-29 [PATCH] vfs: add FS_USERNS_DELEGATABLE flag and set it for NFS Needs Work LLM Accepted

This patch adds a new flag FS_USERNS_DELEGATABLE to the Linux kernel's file system (fs) module, which allows certain file systems like NFS to delegate user namespace handling to their parent namespaces. The patch sets this flag for NFS and resolves an issue where containerized environments were unable to mount NFS shares due to a recent change that restricted non-init user namespaces from mounting file systems without the FS_USERNS_MOUNT flag set.
